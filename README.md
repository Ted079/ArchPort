# ArchPort
ArchPort is a full-stack MERN platform for showcasing architectural, interior, and landscape projects.
Designed as a learning platform where architecture students can upload and share their work.

Status: In active development.
Features (current):
User registration & login, JWT-based authentication,
Create / edit / delete projects (CRUD),
Upload project images & user avatars via Cloudinary,
Project filtering & search,
User profiles,
View counter,
Form validation with Zod + React Hook Form

Tech Stack.
Frontend:
React + TypeScript (Vite),
Redux Toolkit / Async Thunks / rtk query,
React Router,
React Hook Form + Zod,
Tailwind CSS,
Axios

Backend:
Node.js + Express + TypeScript,
MongoDB (via MongoDB Atlas),
Mongoose,
JWT Authentication,
Multer + Cloudinary
bcrypt

Project Goal:
This project was built to:
Practice full-stack development (MERN),
Learn backend basics (Express + MongoDB),
Improve TypeScript skills on both client & server,
Work with file uploads and image storage,
Apply real-world validation patterns (Zod),
Understand authentication flows,
Build reusable frontend architecture,
